35 ==============================================================================
36 INTRODUCTION                                    vim-efm-langserver-settings-introduction
37 
38 vim-efm-langserver-settings is a Vim plugin that auto-enable and work for
39 efm-langserver (https://github.com/mattn/efm-langserver) under lsp clients.
40 
41 This plugin are under development inspired by mattn/vim-lsp-settings
42 (https://github.com/mattn/vim-lsp-settings).
43 
44 Latest version:
45 https://github.com/tsuyoshicho/vim-efm-langserver-settings.vim
46 
47 
48 ==============================================================================
49 USAGE                                           vim-efm-langserver-settings-usage
50 
51 Installing under plugin-manger usage.
52 
53 Now, support lsp clients below:
54 
55 - vim-lsp (https://github.com/prabirshrestha/vim-lsp)
56 - ale (https://github.com/dense-analysis/ale)
57 - coc.nvim (https://github.com/neoclide/coc.nvim)
58 
59 See README for supported commands.

69 SETTING                                         vim-efm-langserver-settings-setting
70 
71 g:efm_langserver_settings#debug                 g:efm_langserver_settings#debug
72     Type: Number
73     Default: 0
74 
75     When set 1 to enable efm-langserver's log-file output.
76     Currently log file is fixed file at "~/efm-langserver.log".
